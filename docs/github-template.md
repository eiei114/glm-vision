# GitHub template alignment

This repository was created from the Pi extension template. Use the template as the baseline for README structure, documentation set, and workflows.

## Template reference

- https://github.com/eiei114/pi-extension-template
- Template README structure and badge set

## Keep in sync

- README section order and required badges
- Required docs in `docs/`
- Trusted Publishing workflows (`ci.yml`, `publish.yml`, `auto-release.yml`)
- `package.json` metadata (`name`, `repository`, `homepage`, `bugs`)
- Repository settings (see `repository-settings.md`)

## Updating from template

1. Compare the current README with the template README.
2. Update docs and badges to reflect any new template requirements.
3. Confirm workflows still match Trusted Publishing rules.
4. Update `docs/template-checklist.md` when changes land.
